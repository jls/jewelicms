Using mod_rewrite to return default images

Posted 3 days ago

The Problem

We have a web app running that displays pages of thumbnails. Instead of having a rails action that would return the thumbnail or a default image if no thumbnail was available, I used Apache’s mod_rewrite to do it for me.

The behavior I was looking for was if the image requested existed then pass the request through. If the image requested did not exist on the file system then rewrite the request to /images/image_not_available.jpg
